# Template – Fullstack Node.js + React

## Purpose
Full-stack project template with backend and frontend separated.

## Stack
- Backend: Node.js + Express + Socket.IO
- Frontend: React
- Communication: REST + WebSocket

## Project Structure
- `backend/` → API & Socket.IO server
- `frontend/` → React application
- `docs/` → Deployment documentation

## Getting Started

See [Deployment Guide](docs/DEPLOYMENT.md)

## Usage
1. Click **Use this template**
2. Select Owner (destination organization)
3. Select Repository Name

## Notes
Template-only repository. No production code.
